## 使用

### 1. 安装cyndi
npm i cyndi

### 2. litten依赖的库
cyndi，依赖下面的库。
1. [react](https://github.com/facebook/react)
2. [lodash](https://github.com/lodash/lodash)

## 功能

### getPrefixNs
获取控件css前缀

### isEmptyString
检测是否是空字符串

### printArrayItem
遍历对象数组，将对的每个属性输出成字符串，不递归遍历对象属性的子属性
